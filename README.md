中選會網站資料Parse
================================================

[president/president.json]：總統得票數資料(至開票所等級)

[mapping]：各選區與中選會網站的網址對應

中選會網站不分區立委各地區各政黨得票數(率) 資料
================================================

得票資料格式為：
```
name: xxx
link: 中選會網站網址
data: 中選會網站表格轉json
nodes: 各子分區的得票資料，格式同得票資料
```
